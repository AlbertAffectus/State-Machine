# affectus-neuron

## Preqreqs

.env with

```
PORT
MONGODB_URI
OPENAI_API_KEY
PINECONE_API_KEY
PINECONE_TOPICS_INDEX
```

spin up mongodb

## Run the thing

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run
OR
bun run dev
```

This project was created using `bun init` in bun v1.1.29. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
